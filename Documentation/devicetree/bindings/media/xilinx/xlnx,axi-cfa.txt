Xilinx Color Filter Array (CFA)
-------------------------------

Required properties:

- compatible: Must be "xlnx,axi-cfa".

- reg: Physical base address and length of the registers set for the device.

- xlnx,axi-video-width: Video pixel component width, as defined in video.txt.

- ports: Video ports, using the DT bindings defined in ../video-interfaces.txt.
  The cfa as an input port (0) and and output port (1).

Example:

	axi_cfa_0: axi_cfa {
		compatible = "xlnx,axi-cfa";
		reg = <0x400b0000 0x10000>;

		xlnx,axi-video-width = <8>;

		ports {
			#address-cells = <1>;
			#size-cells = <0>;

			port@0 {
				reg = <0>;
				cfa0_in: endpoint {
					remote-endpoint = <&spc0_out>;
				};
			};
			port@1 {
				reg = <1>;
				cfa0_out: endpoint {
					remote-endpoint = <&ccm0_in>;
				};
			};
		};
	};
