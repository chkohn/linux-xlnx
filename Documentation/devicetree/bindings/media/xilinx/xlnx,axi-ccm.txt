Xilinx Color Correction Matrix (CCM)
------------------------------------

Required properties:

- compatible: Must be "xlnx,axi-ccm".

- reg: Physical base address and length of the registers set for the device.

- xlnx,axi-video-format, xlnx,axi-video-width: Video format and width, as
  defined in video.txt.

- ports: Video ports, using the DT bindings defined in ../video-interfaces.txt.
  The ccm has an input port (0) and an output port (1).

Example:

	axi_ccm_0: axi_ccm {
		compatible = "xlnx,axi-ccm";
		reg = <0x400c0000 0x10000>;

		xlnx,axi-video-format = "rgb";
		xlnx,axi-video-width = <8>;

		ports {
			#address-cells = <1>;
			#size-cells = <0>;

			port@0 {
				reg = <0>;
				ccm0_in: endpoint {
					remote-endpoint = <&cfa0_out>;
				};
			};
			port@1 {
				reg = <1>;
				ccm0_out: endpoint {
					remote-endpoint = <&gamma0_in>;
				};
			};
		};
	};
