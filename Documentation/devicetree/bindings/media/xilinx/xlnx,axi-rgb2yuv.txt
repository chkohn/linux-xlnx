Xilinx RGB to YUV (RGB2YUV)
---------------------------

Required properties:

- compatible: Must be "xlnx,axi-rgb2yuv".

- reg: Physical base address and length of the registers set for the device.

- xlnx,axi-input/output-video-format, xlnx,axi-video-width: Video format and
  width, as defined in video.txt

- ports: Video ports, using the DT bindings defined in ../video-interfaces.txt.
  The rgb2yuv has an input port (0) and an output port (1).

Example: RBG to xRGB rgb2yuv

	axi_rgb2yuv_0: axi_rgb2yuv {
		compatible = "xlnx,axi-rgb2yuv";
		reg = <0x40100000 0x10000>;

		xlnx,axi-video-width = <8>;

		ports {
			#address-cells = <1>;
			#size-cells = <0>;

			port@0 {
				reg = <0>;
				rgb2yuv0_in: endpoint {
					remote-endpoint = <&gamma0_out>;
				};
			};
			port@1 {
				reg = <1>;
				rgb2yuv0_out: endpoint {
					remote-endpoint = <&cresample0_in>;
				};
			};
		};
	};
